# Gateway

Yêu cầu: python3, pip

Dùng pip để cài đặt thư viện cần thiết

```commandline
pip install -r requirements
```

Trong thư mục env.py, thay đổi các thông tin của AIO_USERNAME, AIO_KEY, AIO_FEED_KEYS tương ứng với thông tin trên
Adafruit server 

Khởi chạy gateway

```commandline
python main.py
```
